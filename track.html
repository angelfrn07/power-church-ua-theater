How this works

  1. Pick a recipient (or type a custom name).
  2. Pick which page or PDF you want them to land on.
  3. Click "Copy URL" and paste it into iMessage or email.
  4. When they click, the visit gets logged on the stats page tagged with that name — so you can see who's read what.

Privacy note: the link doesn't carry any personal info — just the tag you set. The recipient never sees the tag in their browser bar's title. The counter is anonymous and stores only an integer count per tag — no IP, no name, no cookies.
